About

Cool Simple Site is a handy resource for modern styling that uses only HTML and CSS - no minified JS, no confusing framework to learn, no difficult class/ID overrides to change colors or spacing.

Cool Simple Site styles your html elements with just one stylesheet file; modern styling doesn't get any more lightweight than that!

Instructions

  1. First download the stylesheet
  2. Add the stylesheet to your project directory
  3. Reference the stylesheet in your document head 
    
                <link rel="stylesheet" href="./styles.css"</link>
  4. Copy and paste elements to add them to your website!
